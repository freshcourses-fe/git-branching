Ветка - указатель на комит

HEAD - указатель на ветку 
Оторваная Голова - HEAD который указывает на комит а не ветку.

Создание ветки:
  Находясь на комите на котоырй ветка изначально будет куазыаать выполнить одну из команд
  git checkout -b <имя_ветки>
  git branch <имя_ветки>
  git switch -c <имя_ветки>

Просмотр веток:
  git branch

Удаление веток:
  git branch -d <имя_ветки>


Слияние веток:
  Находясь в ветке в которую будут сливатся изменения ввести git merge <имя_ветки>
  Если конфликтов не было то увидите надпись fast-forvard и болше ничего делать не надо
  
  Если были конфликты между сливаемыми ветками, то их прийдется решить
  - можно принять изменения из текущей ветки
  - можно принять изменения из сливаемой ветки
  - можно принять изменения из обоих веток
  После этого надо будет проиндексировать файлы в которых были конфликты и произвести комит.
  У комита уже будет имя по умолчанию.

  ТАкже можно отказатся от проведения слияния командой git merge --abort
